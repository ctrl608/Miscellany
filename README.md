# 序

乙巳岁腊月, 余居于寝, 游乎网络; 目接万象, 意骋八荒. 见夫人情, 托之模因; 悲欢离合, 寄以表情: 或一笑释忧怀, 或片言见微旨. 予独爱其机锋不刻, 言有欢而意无穷. 顷刻潮起, 倏忽湮沉, 观其浮沉网海, 恐散佚如烟, 乃随见随录, 择善而藏.

先是四五载, 寒暑递嬗, 搜罗未辍. 予手自更名, 以正其称; 亲加区分, 以定其部. 或夜分得之, 辄记其目; 或偶有所会, 复订其名. 得一图, 则辨其类属; 成一编, 则志其增损. 稿凡数易, 目屡重排; 去取反复, 增删再三. 积久乃繁, 散帙复整.

及其卷帙既富, 条贯粗备, 始念: 零藏私箧, 终虞散失; 独玩一隅, 未免湮没. 于是创为一仓, 建库云端, 汇诸旧录, 悉数上载; 每定一编, 则志其一变; 使聚散有归, 检索有序, 开卷可览, 随取随得.

夫是等图像, 世或目为戏玩, 而余谓其中自有风神: 远可以见世相, 迩可以寓人心. 临众生之百态, 留万代以瞻观; 后之有知, 或亦赖此数卷, 以观昔年其嬉笑怒骂哉!

是为序.
